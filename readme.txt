Весь проект нужно сохранить в произвольной папке(например: the_directory)
Структура должна выглядеть следующим образом:
the_directory
    pythoncode <- директория
    app.py
    start.sh
    stop.sh
    commands.txt <- команды, которые использовал в процессе
    readme.txt
    skazka.txt <- отсюда в рамках проекта "краулил" слова

start.sh <- сказивает python3-venv и создает его
все библиотеки качает в виртуальное окружение

скачивает кафку архивом -> распаковывает -> убирает архив
-> запускает zookeeper, kafka broker -> создает topic

Запускает контейнер, который будет удален при установке, с Postgres
И внутри postgres создает необходиму таблицу, которая будет принимать "слова"

Далее запускает Flink job и task менеджеры
Использовать flink, к сожалению не сумел.

Последними шагами запускает 2 python скрипта в фоне
и элементарный Flask веб сервер доступный по адресу localhost:5000

Все процедуры запускал в 1 терминале, который в итоге будет держать Flask
Запрос отправлял с 2 терминала следующим образом:
curl localhost:5000/?word=дом <- скрипт вернет 10 наиболее похожих слов из базы
Сама база наполняется словами пока скрипт to_flask читает сказку о золотой рыбке.

после закрытия сервера Flask
stop.sh убьет все, что было создано, вернув в исходное состоянии
